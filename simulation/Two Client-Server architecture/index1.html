Instructions

  • 1- Send Request: Click the "Send Request" button to enter your search query in the text area that appears, then press Enter to initiate sending data packets from the client nodes to the server.
  • Note: The user can only ask the following questions:
    • What is client?
    • What is client-server architecture?
    • What is server?
    • What is internet?
    • What is computer network?
  • 2- Send Response: After sending the request, click the "Send Response" button to process and visualize the server response.
  • The response will be generated based on predefined FAQs and displayed in the response text area.
  • The packets will move from the server back to the client nodes, indicating the response flow.
  • 3- Reset: Click the "Reset" button to refresh the page and reset all animations and data.
  • 4- Print: Click the "Print" button to print the current state of the page.
